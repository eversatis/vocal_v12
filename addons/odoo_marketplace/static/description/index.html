Odoo Multi Vendor Marketplace
Multi Seller/Vendor Option For Your ODOO E-Commerce
Convert Your Odoo Store Into An Online Multi Vendor Marketplace

Installation and Configuration Click here for the video tutorial User Guide

Features:

  • Multi seller/vendor support
  • Seller's commission management
  • Various invoice generation options
  • Separate seller's product collection and management
  • Marketplace product stock management
  • Vendor / Seller and Admin moderation and approval
  • Global and Different seller commission for every seller
  • Option to view the list of all the sellers of the marketplace at one place
  • Seller information on product page
  • Seller profile link and seller shop link on product page
  • Separate seller profile page for every seller
  • Separate seller shop page for every seller
  • Seller reviews on seller profile page
  • Admin can also sell its own products on store
  • Admin can add terms and condition on seller registration form
  • Admin can set manual or auto approval of seller request
  • Admin can set manual or auto approval of seller products
  • Admin can set manual or auto approval of seller product's quantity
  • Only sellers approved by admin can upload their products on store
  • Only approved products by admin are displayed on store
  • Admin can easily convert their site into complete advanced marketplace
  • Admin can link their profile to their social media account
  • Admin can set marketplace currency for seller payment

Website View

Sell Option on Website

Sell option redirects on sell page to seller signup.

Sell Page

Sellers are register through this page.

Seller Registration Form Page

On this seller will fill required details to become a seller.

Marketplace Menu For Seller

By clicking on "Marketplace" option seller can reach "Seller Dashboard" menu at backend.

Seller Informations on Product Page

Customer can see seller information and seller profile. Customer can also see the shop details of seller.

Seller Profile Page

This is a seller profile page, here anyone can see all information related to seller.

Seller Reviews on Seller Profile Page

Under Rating & Reviews tab, all the reviews for the seller will display.

Seller Shop Page

This is a seller shop page, all the products of seller and other details will display here.

All Seller List

On clicking here, customer can see list of all sellers of marketplace.

Seller Side View

Seller Signup Form

Seller will signup using this form on website.

Marketplace Dashboard For Seller

Seller can see their marketplace products, orders, and payments count at one place.

Seller Dashboard For Sellers

Seller Dashboard with less menu when seller is not approved.
Seller Dashboard with more menu when seller is approved.

Seller Profile

Seller Can See His/Her Profile.

Seller Shop

Seller can see his/her shop and manage it.

Orders Of Seller's Product

Seller Can See Orders of His/Her Product.

Seller Own Payments

Seller can see all the payment corresponding to his/her products.

Seller Own Product

Seller can create product and manage it under product menu..
Seller can see all the approved products here.
Seller can see all the non-approved products here.

Stock/Inventory Request By Seller

Seller can request for inventory/stock under inventory request menu.

Delivery Order For Seller's Products

Seller can see status of delivery order for his/her product order.

Stock Moves of Seller's Products

Seller can see all the stock moves of his/her product.

Admin Side View

Marketplace Dashboard For Admin

Admin can see marketplace sellers, products, orders, and payments count at one place.

Admin can manage Sellers

Admin can see status of all sellers under Seller menu.
Admin can approve/deny any seller of marketplace.
Admin can change default settings for any seller from here.
Admin can manage marketplace curreny for this seller.

Sellers Shops

Manage seller shop for marketplace sellers.

Sellers Reviews and Ratings

Manage reviews,ratings for marketplace sellers.

Sellers Recommendations

Manage recommendations for marketplace sellers.

Orders Of Marketplace Product

Manage order for marketplace product.

Make Payment to Seller

Admin can pay to seller from here.

Seller Payments

Manage seller payments under seller payments menu.

Seller Bills

Manage seller bills under seller bills menu.

Order Analysis

Admin can perform the analysis of Sales Orders of marketplace.

Seller Product

Manage Seller products under products menu.
Admin can see all the Non-Approved products of all Seller.
Admin can see all the Approved products of all Seller.

Stock/Inventory Request

Manage inventory request under inventory request menu.

Marketplace Delivery Orders

Manage delivery orders for marketplace orders.

Marketplace Stock Moves

Admin can see all the Stock Moves of products of marketplace here.

Configuration Settings For Marketplace
