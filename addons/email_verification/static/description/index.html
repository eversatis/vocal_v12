Email Verification

Send token based emails to the customers on signup. Restrict checkout for unverified accounts

Salient features
  • Send token based emails for verification to the customers while signup.
  • Set the validity of the tokens in the configuration by your own.
  • Accounts will not be activated until email is verified.
  • On clicking the link sent to the mail you will be redirected to a simple page.
  • Restrict unverified users from checkout.
  • Resend email if the verifying link is expired or user wants a new email.
  • Automatic redirection to login after clicking on the valid email link.

Basic Requirements

There should be dbfilter and db_name used in the config file of your server.
