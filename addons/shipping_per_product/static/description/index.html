Shipping Per Product
This module allows customer to select delivery method for each product separately and delivery amount will be calculated for each product.

Features:

  • Admin can add different delivery methods for each product.
  • Customer can choose a delivery method for each product and delivery price will be calculated for each product separately.

  • Work Flow

Work Flow

Set Delivery Methods For Product

Admin can add different delivery methods for each product.
These delivery methods will be available on website for this product from which a user can choose only one.
Admin can also add product(s) in a particular delivery method.
That delivery method will be available for the added product(s) on website.

Website View

Customer can choose delivery per product before payment.

Backend View

"Dynamic Delivery Charges" is a delivery method that will be applicable if you want to apply shipping charges per sale order line i.e. shipping charges per product.
While Creating quotation from backend admin can choose delivery method for each product separately or can choose a single delivery method for sale order.
For shipping per product admin has to select "Dynamic Delivery Charges" as a delivery method in quotaion. If he choose other delivery method then that delivery method will be applicable for sale order.
Admin can see delivery methods and delivery charges for each sale order line.
After order is confirmed delivery will be generated separately for each sale order line.
