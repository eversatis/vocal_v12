What's Elasticsearch?

Elasticsearch is a search engine based on Lucene. Elasticsearch is the fastest search engine that returns the results for the entered query in real-time. Elasticsearch is actually a highly scalable open-source full-text search and analytics engine. It allows you to store, search, and analyze big volumes of data quickly and in near real time.

Why Elasticsearch ?

Fast, Incisive Search against Large Volumes of Data.

Are you running an online web store where you are allowing your customers to search for products that you sell? If yes, you can use Elasticsearch to store your entire product catalog on the server where you have installed the Elasticsearch and provide search and autocomplete suggestions for the customers of your store.

Suppose, that we are searching a word "Apple" on our local SQL database. It will surely be a time-consuming process as all the data will be checked through for finding the word "Apple". But, if we make use of the Elasticsearch and put all of our catalog data in the Elasticsearch server it would for sure return us with the IDs' of the products having the "Apple" word associated with them. Thus narrowing-down the search results for us and in turn providing faster search results.

Manage your index with ease

After the successful installation of this extension on your web store, the admin will navigate through Elasticsearch ->Elastic Index Mapping -> Select Index Name ( product-template) . Here, you can create/update and delete(reset) the index in elastic search server.

You can forcely update the index by button or schedule a cron for syncing the product information, for cron the admin will navigate through Elasticsearch -> Configuration -> Elastic Setting -> go to the menu Cron Setting .

Manage your search dropdown

After creating indexes you can easily manage your search dropdown on enabling the product Suggesstion in Website Settings, you can easily show the search result on entering in search menu of website and you can customize the dropdown like limit the search result, show product thumbnail and limit the description character.

In Advance settings you can manage the start searching character in website search menu and resize the menu size for dropdown .

Manage your Elastic Server with Website

To connect your Elastic server with your web store, the admin will navigate through Elasticsearch -> Configuration -> Select Connection Setting (connection) . Here, you have to enter the URL and the Port number of the Elastic server to connect with your store .
