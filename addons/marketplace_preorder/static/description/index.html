Odoo Marketplace Pre-Order
This module allow customers to pre-order the seller's out of stock products

Note: This module is an addon for Odoo Multi Vendor Marketplace module. The price reflected here includes all dependent module's cost as well.

Features:

  • Seller can increase their sale by preordering their products.
  • Seller can set out of stock products for preorder.
  • Seller can set either full amount or percentage of amount to be paid for preorder.
  • Seller can set minimum and maximum quantity that can be preorder by customer.
