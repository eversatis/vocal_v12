Odoo DHL Shipping Integration
Professional Comprehensive DHL Shipping Tool For Odoo E-COMMERCE

While using the module please ensure these things:

  • Valid Credentials.
  • None-Zero Amount Of Weight.
  • Valid Address For User/Partner [Street, City, Zip, State Code, Country Code, Phone ].
  • Valid Address For Warehouse [Street, City, Zip, State Code, Country Code, Phone ].

This Extension Allows

  • Real-time Shipping Rate Calculation By Admin.
  • Real-time Shipping Rate Calculation By User.
  • For Both Domestic & International.
  • Automatically Determination of Domestic Express(N),European Wide(EU) as well as International Express(P) Service .
  • Shipping Label Generation.
  • Option Of Enable/disable, Edit The Names and Add Service Costs On Shipping Services.
  • Live Shipment Tracking Number .
  • Compatible With All Version Of Odoo.
  • Full Admin Control
    • Admin can Add Extra Service Charges on actual shipping price .
    • Admin can Define/Update New Service as per their preferences .
    • Admin can Define/Update Packaging Type and Drop Off Type for DHL shipping.
    • Admin can Define/Update Allowed Shipping Methods and Unit of Weight for DHL shipping.

Setup & Configure

Installation

This module required the extension of BackBone Shipping Module.

Shipping Configuration .

After the successful installation of the DHL Shipping Module module you will have to click the Delivery Methods menu under Inventory ->Delivery->Delivery Methods.

We have set up the most frequently used values
But please look all of them and change according to your needs .

Fill these DHL Credentials

  • Site ID.
  • Account Number.
  • Password.

Environment - Select "Test Environment" while testing the module flow. Turn into "Production Environment", when you go live and ready to create real billable labels.

Work-flow

DHL Price Calculator [On BackEnd By Admin]

After configuring the DHL Shipping Module you will be able to see the DHL Delivery in Delivery Method List Of Sale Order Option [Sale --> Sale Order->Create].

For Using DHL Price Calculator

  • Select A customer
  • Add product in sale oder line
  • Select DHL Delivery as Delivery Method for this sale order .
  • Click on Set Price to add this delivery charges to the sale order.

DHL Price Calculator [On FrontEnd By Site User]

This module also support the feature of Price Calculation From Website .

On the ODOO E-Commerce the user can see the DHL Delivery option in Delivery Method List In their cart before making final payment .

DHL Label Generator

Prior to label generation enable packaging option and assign access of packaging.

NOTE: Firstly you will have to create package of picking before clicking the validate.

Now create package by selecting those picking line which you want to put in single package then click on put in pack .

In side the wizard select the DHL Package Type you want to use for this package.

Here you can also modify the package Shipping Weight and also add the cover amount and description of package for future reference.

Finally After package generation click on validate/send to shipper.
