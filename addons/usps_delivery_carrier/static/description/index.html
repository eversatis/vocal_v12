Odoo USPS Shipping Integration
Odoo E-Commerce Extension For Providing You Real-Time Shipment With USPS Delivery Service

While using the module please ensure these things:

  • Valid Credentials.
  • None-Zero Amount Of Weight.
  • Valid Address For User/Partner [Street, City, Zip, State Code, Country Code, Phone, Email ].
  • You should be a valid USPS eVS customer (Since this module uses USPS eVS Labl API for generating shipping labels)

This Extension Allows

  • Real-time Shipping Rate Calculation By Admin.
  • Real-time Shipping Rate Calculation By User.
  • Automatic Packaging Options [Optimal Solution].
  • Manual Packaging Options
  • Shipping Label Generation.
  • Multi Package Shipment.
  • Live Shipment Tracking Number .
  • Compatible With All Version Of Odoo.
  • Full Admin Control
    • Admin can Add Extra Service Charges on actual shipping price .
    • Admin can Define/Update New Service as per their preferences .
    • Admin can Define/Update Packaging Type for USPS shipping.
    • Admin can Define/Update Allowed Shipping Methods and Unit of Weight for USPS shipping.

Setup & Configure

Installation

This module required the extension of BackBone Shipping Module.

After the successful installation of the BackBone Shipping Module module you will be able to see USPS Shipping Service Name under the Setting ->General Settings ->Shipping Service.

USPS Credentials

After the installation you have to set the credentials from Delivery Methods.

Fill these USPS Credentials

  • User ID - Valid USPS User ID

  • Environment - Select Testing Shipping while testing the module flow. Turn into Production, when you go live and ready to create real billable labels.

Shipping Configuration .

We have set up the most frequently used values
But please look all of them and change according to your needs .

Work-flow

USPS Price Calculator [On BackEnd By Admin]

After configuring the USPS Shipping Module module you will be able to see the USPS Delivery in Delivery Method List Of Sale Order Option [Sale --> Sale Order->Create].

Now :

  • Select a customer .
  • Add product in sale order line .
  • Select USPS Delivery as delivery method for this sale order .
  • Click on Get rate it will list you the delivery charges.
  • Now Click on Add to order it assign delivery charges in sale order.

USPS Price Calculator [On FrontEnd By Site USer]

This module also support the feature of Price Calculation From Website .

On the odoo website user can see the USPS delivery option in delivery method List in their cart before making final payment .

USPS Label Generator

NOTE:

You must be a valid eVS Label API customer (Since this module uses USPS eVS Labl API for generating shipping labels)
Prior to label generation --> enable packaging option and assign access of packaging to user .

NOTE: Firstly you will have to create package of picking before clicking the validate.

Now create package by selecting those picking line which you want to put in single package then click on put in pack .

In side the wizard select the USPS Package Type you want to use for this package.

Here you can also modify the package Shipping Weight and also add the cover amount .

Finally After package generation click on validate/send to shipper.

Now download the label from the message box.
