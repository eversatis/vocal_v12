Odoo FedEx Shipping Integration
Odoo E-Commerce Extension Providing In Depth Integration With FedEx

While using the module please ensure these things:

  • Valid Credentials.
  • None-Zero Amount Of Weight.
  • Valid Address For User/Partner [Street, City, Zip, State Code, Country Code, Phone ].
  • Valid Address For Warehouse [Street, City, Zip, State Code, Country Code, Phone ].

This Extension Allows

  • Real-time Shipping Rate Calculation [From BackEnd as well Website Also].
  • Automatic Packaging Options [Optimal Solution].
  • Manual Packaging Options .
  • Shipping Label Generation.
  • Multi Package Shipment.
  • Shipping Label Cancellation.
  • Live Shipment Tracking Number .
  • Compatible With All Version Of Odoo.
  • Full Admin Control
    • Admin can Define/Update New Service as per their preferences .
    • Admin can Define/Update Packaging Type and Drop Off Type for FedEx shipping.
    • Admin can Define/Update Allowed Shipping Methods and Unit of Weight for FedEx shipping.

Setup & Configure

Installation

This module required the extension of BackBone Shipping Module.

After the successful installation of the BackBone Shipping Module module you will have to click the FedEx Shipping Module Link under Setting ->General Settings ->Shipping Service.

Shipping Configuration .

After the successful installation of the FedEx Shipping Module module you will have to click the Delivery Methods menu under Inventory ->Delivery->Delivery Methods.

We have set up the most frequently used values
But please look all of them and change according to your needs .

Fill these FedEx Credentials

  • Key.
  • Password.
  • Account Number.
  • Meter Number.
  • Integration Id.

Environment - Select Testing Shipping while testing the module flow. Turn into Production, when you go live and ready to create real billable labels.

Work-flow

FedEx Price Calculator [On BackEnd By Admin]

After configuring the FedEx Shipping Module module you will be able to see the FedEx Delivery in Delivery Method List Of Sale Order Option [Sale --> Sale Order->Create].

Now:

  • Select A customer .
  • Add product in sale oder line .
  • Select FedEx Delivery as delivery method for this sale order .
  • Click on Set Price it will list you the delivery charges.

FedEx Price Calculator [On FrontEnd By Site User]

This module also support the feature of Price Calculation From Website .

On the odoo website user can see the FedEx delivery option in delivery method List in their cart before making final payment .

NOTE:
This delivery charge is combination of actual delivery charges and percentage/fixed amount set in FedEx delivery configuration .

FedEx Label Generator

NOTE:

Prior to label generation enable packaging option and assign access of packaging.

NOTE: Firstly you will have to create package of picking before clicking the validate.

Now create package by selecting those picking line which you want to put in single package then click on put in pack .

In side the wizard select the FedEx Package Type you want to use for this package.

Here you can also modify the package Shipping Weight and also add the cover amount and description of package for future reference.

Finally After package generation click on validate/send to shipper.

Now down load the label from the message box.

Void Shipment

This module also provide the feature of void shipment .
thus after generating the label if you don't want to ship now then just click on cancellation link to void shipment .
